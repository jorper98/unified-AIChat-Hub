# Unified Chat Hub

## Version 0.0.8

### About
Unified Chat Hub is a self-hosted, private AI development workspace and model playground built with Next.js 14, Tailwind CSS, and MongoDB. It provides a unified interface for interacting with multiple AI models through OpenRouter, with features like thread management, system prompt customization, and real-time token/cost tracking.

### Features
- Multi-model support via OpenRouter integration
- Persistent chat history with search functionality
- Custom system prompts with save/load capabilities
- Real-time token estimation and cost tracking
- Dark/light theme support
- Thread archiving and management
- Markdown rendering with syntax highlighting

### Technical Stack
- **Frontend**: Next.js 14.1.4, React 18.2.0, TypeScript 5.4.3
- **Styling**: Tailwind CSS 3.4.1
- **Database**: MongoDB 6.5.0
- **API Integration**: OpenRouter
- **Container Support**: Docker
